# ✏ Air Drawing — Desktop App

## Quick Start

```bash
# Step 1 — Install dependencies
pip install -r requirements.txt

# Step 2 — Launch the app
python launcher.py
```

## Files

| File | What it is |
|------|-----------|
| `launcher.py` | Desktop GUI — run this |
| `air_drawing.py` | YOUR original code — untouched |
| `requirements.txt` | Dependencies |

## Gestures (from your code)

| Gesture | Action |
|---------|--------|
| ☝ Index finger | Draw |
| ✌ Two fingers | Hover / stop drawing |
| ✊ Fist | Pause |
| 👍 Thumb up | Undo |
| 🖐 All 4 fingers (hold 1.5s) | Clear canvas |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `[` / `]` | Brush size down / up |
| `Z` | Undo |
| `C` | Clear |
| `1`–`9` | Quick color select |
| `Q` / `ESC` | Quit |
